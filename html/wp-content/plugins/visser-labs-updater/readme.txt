== Description ==

Install the Visser Labs Updater Plugin to get update notifications for your Visser Labs Plugins.

== Installation ==

1. Upload the folder 'visser-labs-updater' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

Just sit back and relax - Visser Labs Updater will automatically check for updates and notify you when they are available.
